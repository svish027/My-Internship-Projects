                                                            (Weather App)

A simple and responsive Weather App built using HTML, CSS, and JavaScript that allows users to check real-time weather information for any city in the world.

Features:-

Search by City: Get current weather details for any location.
Real-Time Data: Displays temperature, humidity, and wind speed using a weather API.
Dynamic UI: Automatically updates with weather icons and background changes based on conditions.

Responsive Design: Works seamlessly on desktop and mobile devices.
Error Handling: Alerts the user if an invalid city is entered.

Technologies Used:-

HTML5 – For the structure of the app.
CSS3 – For styling and responsive layout.
JavaScript (ES6) – For fetching data and dynamic content updates.
OpenWeatherMap API – For real-time weather data.

How It Works:-

The user enters a city name in the search bar.
The app fetches weather data from the API using fetch() in JavaScript.
Weather details (temperature, condition, humidity, etc.) are displayed dynamically on the page.

Future Improvements:-

Add 5-day weather forecasts.
Implement dark/light mode.
Add geolocation to detect user’s current location automatically.

                                                           (TO-DO-APP)

To-Do List App:-

A simple and interactive To-Do List Application built using HTML, CSS, and JavaScript. This app helps users 
manage daily tasks efficiently by allowing them to add, edit, mark as complete, and delete tasks — all within 
a clean and responsive interface.

Features:-

Add Tasks: Easily add new tasks to your list.
Edit Tasks: Update task details anytime.
Mark as Completed: Check off tasks once they’re done.
Delete Tasks: Remove completed or unnecessary tasks.
Local Storage Support: Tasks are saved in the browser, so they remain even after refreshing the page.
Responsive Design: Works smoothly on desktop and mobile devices.

Technologies Used:-

HTML5 – Structure of the app
CSS3 – Styling and layout
JavaScript (ES6) – Functionality and interactivity
Local Storage API – For persistent task saving

How to Use:-

Clone or download this repository.
Open index.html in your browser.
Add your tasks and start organizing your day!

Recipe Book App:-

The Recipe Book App is a simple and interactive web application built using HTML, CSS, and JavaScript. It 
allows users to browse, search, and view various recipes with detailed ingredients and cooking instructions
all in one place.                                                         

Features:-

View Recipes: Display a collection of delicious recipes with images and descriptions.
Search Functionality: Easily find recipes by name or ingredients.
Recipe Details: View detailed cooking steps, ingredients, and preparation time.
Local Storage Support: Save favorite recipes for later access.
Responsive Design: Works seamlessly across desktops, tablets, and mobile devices.

Technologies Used:-

HTML5 – Structure of the web app
CSS3 – Styling and responsive design
JavaScript (ES6) – Dynamic content and interactivity

How to Use:-

Clone or download this repository.
Open the index.html file in your browser.
Browse available recipes or search for a specific one.
Click on a recipe to view its details.

                                                          (Interactive Quiz)

Interactive Quiz App:-

An Interactive Quiz Application built using HTML, CSS, and JavaScript.
This project allows users to test their knowledge through a series of multiple-choice questions. It features 
dynamic question rendering, score tracking, and real-time feedback to create an engaging quiz experience. 

Features:-

Multiple-choice questions with instant feedback
Score tracking and result display
Ability to restart the quiz
Responsive design for mobile and desktop

Technologies Used

HTML5 – Structure of the app
CSS3 – Styling and layout (responsive design)
JavaScript (ES6) – Quiz logic, DOM manipulation, and interactivity

How It Works:-

The user starts the quiz by clicking the “Start” button.
Each question is displayed one by one with multiple answer options.
The user selects an answer, and the app immediately shows whether it’s correct.
The final score is displayed at the end with an option to retake the quiz.
